# Structure Notes

تم التقسيم بطريقة محافظة جدًا حتى لا تضيع أي ميزة من المشروع.

## ماذا تم؟

- استخراج كل بلوكات CSS إلى ملفات منفصلة داخل `assets/css`.
- استخراج كل بلوكات JavaScript إلى ملفات منفصلة داخل `assets/js`.
- الحفاظ على ترتيب التحميل كما كان في الملف الأصلي.
- الإبقاء على نفس أسماء `id` للـ style/script tags قدر الإمكان لتجنب كسر أي كود يعتمد عليها.

## عدد الملفات المستخرجة

- CSS blocks: 31
- JavaScript blocks: 29

## لماذا لم يتم تحويله إلى ES Modules الآن؟

لأن الملف الأصلي يعتمد على Global Scope وباتشات متتابعة. التحويل إلى Modules مباشرة قد يكسر وظائف تعمل الآن.

المرحلة المناسبة لاحقًا:

1. تثبيت النسخة المقسمة.
2. تشغيل QA داخلي.
3. نقل جزء واحد فقط كل مرة إلى module واضح.

## أين أعدل؟

- الستايل العام: `assets/css/00-mogahed-clean-styles.css`
- السكريبت الأساسي الكبير: `assets/js/00-mogahed-clean-scripts.js`
- إصلاحات V82/V83: ابحث عن ملفات `v82` و `v83` داخل `assets/js` و `assets/css`.
