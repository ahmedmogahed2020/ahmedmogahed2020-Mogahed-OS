# QA Checklist

بعد رفع أو تعديل المشروع، اختبر التالي:

## تشغيل عام

- [ ] الصفحة تفتح بدون شاشة بيضاء.
- [ ] التنقل السفلي يعمل على الموبايل.
- [ ] البحث يفتح ويعرض نتائج.
- [ ] System Health يفتح.
- [ ] Run System Test يعمل.

## البيانات

- [ ] إضافة مهمة.
- [ ] تعديل مهمة.
- [ ] حذف/أرشفة عنصر ثم Undo/Redo.
- [ ] Export Backup.
- [ ] Import Backup على نسخة اختبار فقط.

## المعرفة

- [ ] إضافة فيديو.
- [ ] إضافة كتاب/PDF.
- [ ] فتح المشغل الداخلي.
- [ ] فتح PDF Reader.
- [ ] تكبير/تصغير PDF.
- [ ] تحويل معرفة إلى مهمة/فكرة/هدف/إجراء.

## الطوارئ والداشبورد

- [ ] زر الطوارئ يعمل.
- [ ] خطة الإنقاذ تظهر.
- [ ] الداشبورد لا يختفي.

## GitHub Pages

- [ ] افتح المشروع من GitHub Pages وليس file:// فقط.
- [ ] راقب Console لأي أخطاء.
- [ ] تأكد أن Service Worker لا يعرض نسخة قديمة.
