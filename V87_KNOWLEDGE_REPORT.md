# V87 Knowledge System Upgrade — Stage 4

## Scope
ترقية نظام المعرفة فقط بدون الدخول في الداشبورد الذكية أو نظام المهام/الأهداف المتقدم.

## تم تنفيذه

- إضافة طبقة معرفة V87 آمنة فوق V86.
- صفحة معرفة جديدة تعرض:
  - إحصائيات الكتب والفيديوهات والبودكاست والمكتمل.
  - فلاتر حسب النوع.
  - بحث داخل المعرفة.
  - كروت معرفة احترافية ومناسبة للموبايل.
- إضافة معرفة مختصرة بالحقول الأساسية:
  - الاسم/العنوان.
  - النوع.
  - المجال.
  - الرابط.
  - الإجمالي.
  - وصلت إلى.
  - الغلاف.
  - رفع ملف من الجهاز.
  - ملخص قصير.
- دعم أنواع المعرفة:
  - كتاب / PDF.
  - فيديو.
  - دورة / Playlist.
  - بودكاست.
  - مقال.
- زر جلب البيانات:
  - يحاول ملء العنوان والنوع تلقائيًا من الرابط.
  - يدعم YouTube و Playlist بشكل مبدئي.
  - يستخدم oEmbed لعنوان YouTube عند سماح المتصفح.
- حفظ رابط الفيديو/البلاي ليست/الكتاب داخل المشروع.
- تشغيل الفيديو داخل المشروع باستخدام youtube-nocookie مع تقليل التتبع قدر الإمكان.
- فتح Playlist داخل المشروع باستخدام YouTube embed للـ list.
- إضافة زر إغلاق واضح داخل مشغل المعرفة.
- قارئ PDF للملفات المرفوعة أو روابط PDF.
- إصلاح حفظ تقدم القراءة/المشاهدة.
- أزرار أعلى صفحة الكتاب/المشغل:
  - حفظ التقدم.
  - حوّل لإجراء.
  - تعديل.
  - إغلاق.
- تحويل المعرفة إلى إجراء داخل صفحة الإجراءات.
- تحسين تجربة الموبايل للمعرفة والمشغل والقارئ.

## ملاحظات مهمة

- حجب إعلانات YouTube بالكامل غير مضمون من داخل مشروع ويب عادي. تم استخدام `youtube-nocookie.com` وباراميترات embed لتقليل التتبع والإزعاج قدر الإمكان.
- بعض روابط PDF الخارجية قد لا تظهر داخل القارئ إذا كان الموقع مانع العرض داخل iframe/embed. رفع الملف من الجهاز هو الأكثر ثباتًا.
- جلب بيانات Playlist كاملًا من YouTube يحتاج API Key أو Backend لاحقًا. في V87 تم تنفيذ جلب ذكي مبدئي بدون Backend.
