# دليل استخدام Mogahed OS السريع

## صباحًا

افتح الرئيسية، شوف كارت اليوم، وابدأ بأقرب مهمة واضحة بدل ما تفتح كل الصفحات.

## أثناء اليوم

- أضف مهمة سريعة من زر +.
- لو المهمة لها موعد، ضع تاريخًا ووقتًا واختر التذكير.
- لو وجدت معلومة مهمة، أضفها في المعرفة وحوّلها لاحقًا لمهمة أو مراجعة.
- عند التشتت، استخدم زر الطوارئ.

## آخر اليوم

افتح مراجعة اليوم من الرئيسية أو المراجعات، وسجل ما تم وما يحتاج متابعة غدًا.

## النسخ الاحتياطي

- استخدم تصدير JSON قبل أي تعديل كبير.
- استخدم Google Drive أو Supabase لو تريد نسخة خارج المتصفح.
- استخدم Supabase Storage للملفات الكبيرة.
