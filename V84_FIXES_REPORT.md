# Mogahed OS — V84 Fixes Only

هذه نسخة إصلاحات آمنة فوق V83 بدون إعادة بناء كامل أو إضافة مميزات كبيرة.

## ما تم إصلاحه في V84

- تحسين شاشة البداية/التحميل بشكل أنظف للموبايل.
- دعم تخصيص شاشة البداية من الإعدادات: كلمة مخصصة أو صورة.
- إصلاح سلوك المودالات بحيث يكون الهيدر وزر الإغلاق/الحفظ داخل الكارت ولا يغطي المحتوى أثناء السكرول.
- زر حفظ علوي عام يظهر بجانب زر الإغلاق عند وجود أي زر حفظ داخل المودال.
- إصلاح تموضع البار السفلي وتقليل تغطيته للمحتوى أسفل الصفحات.
- منع زر الثيم/النهار والليل من التحرك كعنصر ثابت فوق المحتوى.
- تحسين كارت وضع التركيز على الموبايل.
- إخفاء شارة V46 Second Brain من الصفحة الرئيسية.
- ضمان وجود زر/مسار الداشبورد في التنقل والـ Quick Dock.
- إضافة fallback للداشبورد عند غيابها حتى تظهر كصفحة كاملة داخل المشروع.
- إضافة handlers مفقودة لأزرار قارئ PDF التي كانت تظهر خطأ Action غير موجود.
- تحسين إشعارات النظام بصريًا اعتمادًا على نظام التوست الحالي.

## حدود هذه المرحلة

هذه المرحلة لا تسترجع كل المميزات الكبيرة مثل:
- الزر العائم نصف الدائرة كاملًا.
- نظام الطوارئ الكامل بـ 150/500 إجراء.
- إعادة بناء صفحات الفوز/المراجعات/المال/العلاقات.
- نظام المهام والأهداف الذكي الكامل.

هذه مخصصة للمراحل التالية V85 وما بعدها.
