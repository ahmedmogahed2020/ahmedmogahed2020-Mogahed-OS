# Checklist قبل رفع Mogahed OS على GitHub Pages

## قبل الرفع

- [ ] فك الضغط عن النسخة.
- [ ] تأكد أن `index.html` في أول مستوى داخل المجلد.
- [ ] تأكد من وجود `css/`.
- [ ] تأكد من وجود `js/`.
- [ ] تأكد من وجود `.nojekyll`.
- [ ] لا ترفع مجلد يحتوي المشروع بداخله كمستوى زائد.

## بعد الرفع

- [ ] افتح رابط GitHub Pages.
- [ ] جرّب الرئيسية.
- [ ] جرّب المهام.
- [ ] جرّب المعرفة.
- [ ] جرّب المزيد.
- [ ] جرّب النسخ الاحتياطي.
- [ ] شغّل System Health.
- [ ] صدّر نسخة JSON احتياطية بعد أول تجربة.

## علامات أن الرفع صحيح

- التصميم ظاهر طبيعي.
- البار السفلي ظاهر.
- زر الإضافة يعمل.
- System Health يعمل.
- لا توجد أخطاء فشل في تقرير QA.
